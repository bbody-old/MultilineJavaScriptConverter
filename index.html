Multiline JavaScript Converter

Description

Converts multiline text into a JavaScript multiline string. This includes both concatenated string in ECMA5, template literal in ECMA6 and a JSON Array.

Uses

  • Large Blocks of text
  • ASCII Art
  • HTML Code
